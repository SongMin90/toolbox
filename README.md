# 万能工具箱 🔧

一个简洁、美观的在线工具箱，集合了多种实用工具，让你的工作效率翻倍！

## ✨ 特性

- 🎨 **精美界面** - 渐变紫色主题，卡片式设计
- 📱 **响应式布局** - 完美适配桌面和移动端
- 🚀 **即开即用** - 纯 HTML 实现，无需安装任何依赖
- 🧰 **多种工具** - 涵盖日常办公、开发、学习等场景

## 🛠️ 工具分类

工具箱包含多个分类，每个分类下有若干实用工具：

- **常用工具** - 日常高频使用的工具
- **开发工具** - 程序员必备
- **办公工具** - 提升办公效率
- **学习工具** - 辅助学习
- **生活工具** - 便民服务

## 🚀 快速开始

### 本地使用

1. 克隆或下载本项目
2. 直接用浏览器打开 `index.html`

```bash
# 克隆项目
git clone <repository-url>

# 打开文件
open toolbox/index.html
# 或者直接在浏览器中拖入 index.html
```

### 在线使用

将 `index.html` 部署到任意静态托管服务：

- GitHub Pages
- Vercel
- Netlify
- Cloudflare Pages

## 📁 项目结构

```
toolbox/
├── index.html    # 主页面（包含所有 HTML/CSS/JS）
└── README.md     # 项目说明
```

## 💡 使用提示

- 点击顶部标签切换不同工具分类
- 鼠标悬停在工具卡片上可查看效果
- 所有工具均在浏览器本地运行，数据不会上传
